# Audio Pitch Identifier

This is my first Machine Learning project built using Python.

The project detects the pitch (fundamental frequency) of an audio file using audio signal processing techniques.

---

#Libraries Used

- numpy
- librosa

(Install them using `pip install numpy librosa`)

---

#How to Run

1. Clone this repository:
   git clone https://github.com/skariachan/Audio_pitch_identifier.git

2. Install required libraries:
   pip install numpy librosa

3. Run the Python file:
   python filename.py
